About

Aidan Sweet is a Game Design student at the University of Montana in Missoula, MT. Some projects that he has worked on so far include an interactive touchscreen game for the Wall Project in the University Center Game Den, he worked as part of the development team for the prototype of a virtual tour mobile application for the university called "Griz Tracks", he is also currently developing a 2D mobile game made in the Unity Engine, and a 3D horror game also made in Unity called "Lost In The Woods". Aidan is also currently learning to create 3D models using the 3D modeling software, Blender.
